# <b>🩺 AI Medical Doctor Assistant</b>

<b>AI-powered Medical Doctor Assistant built using Gradio + Gemini AI.</b>

Analyze uploaded medical images and patient symptoms with a modern healthcare UI and generate professional AI medical-style reports.

---

# <b>🚀 Features</b>

✔ <b>Medical Image Upload</b>  
✔ <b>Patient Symptoms Input</b>  
✔ <b>AI Medical Analysis</b>  
✔ <b>Multiple Medical Departments</b>  
✔ <b>Risk Assessment</b>  
✔ <b>Treatment Suggestions</b>  
✔ <b>Beautiful Medical UI</b>  
✔ <b>Gemini AI Integration</b>

# <b>📦 Installation</b>

## <b>1️⃣ Clone Repository</b>

```bash
git clone https://github.com/your-username/ai-medical-doctor-assistant.git

cd ai-medical-doctor-assistant
```

---

## <b>2️⃣ Install Requirements</b>

```bash
pip install gradio google-genai pillow
```

---

# <b>🔑 Gemini API Setup</b>

Get your API key from:

<b>Google AI Studio</b>

Replace:

```python
API_KEY = "YOUR_GEMINI_API_KEY"
```

with your real API key.

---

# <b>▶️ Run Project</b>

```bash
python app.py
```

OR

```python
demo.launch(debug=True)
```

---

# <b>🧠 Gemini Model</b>

```python
model="gemini-3-flash-preview"
```
